# Example Output with References

## Answer

The biosolids management program in Washington State follows strict regulations [1]. Treatment facilities must comply with both federal 40 CFR Part 503 [2] and state WAC 173-308 requirements [3].

For land application, Class A or Class B pathogen reduction standards must be met [1], and vector attraction reduction requirements apply [4]. The agronomic rate approach ensures nitrogen is supplied appropriately for vegetation growth [5].

Storage facilities must use corrosion-resistant materials [6] and prevent public access [7].
