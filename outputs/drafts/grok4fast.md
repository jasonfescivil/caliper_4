### Workflow for Designing Wastewater Lift Station Flows, Peak Factors, Pumps, and Wetwell for a Town of 400

1. **Determine Service Population and Projected Growth**: Establish current population (400) and project 20-year growth for tributary area, considering future improvements, population increases, and buildout [1] (DOE_Orange_Book_Design_Standards.pdf - p.260; C2-1.2.1 Design Flow Rates). Base on comprehensive plan land use element for consistency [7] (DOE_Orange_Book_Design_Standards.pdf - p.41; G1-3.1 Objective).

2. **Estimate Average Daily Flow (ADF)**: Use per capita flows from Table G2-1 for new systems, assuming normal infiltration; adjust for high groundwater or illicit connections using measured data if available [5] (DOE_Orange_Book_Design_Standards.pdf - p.209; C1-3.3.1 DESIGN FOR AVERAGE DAILY FLOW). For small systems like this, apply 60 gpd/capita (225 L/capita/d) based on household size > national median (2.6 persons) [8] (WEF_FD-16_Facilities_Design.pdf - p.99; 5.3.1 Wastewater Flows). ADF = projected population × per capita flow. Include I/I allowance from actual data for existing conditions [5] (DOE_Orange_Book_Design_Standards.pdf - p.209; C1-3.3.3 INFILTRATION/INFLOW).

3. **Calculate Peak Hourly Design Flow (PHDF)**: Apply peaking factor ≥2.5 from Figure C1-1 (Ratio of Peak Hourly Flow to Design Average Flow) [5] (DOE_Orange_Book_Design_Standards.pdf - p.209; C1-3.3.2 PEAK FLOW). For 400 persons (0.4 thousand), interpolate between 0.3 thousand (≈3.8) and 0.5 thousand (≈3.6) to ≈3.7. PHDF = ADF × peaking factor. Base on 20-year forecast; evaluate minimum flows for low-flow operation [1] (DOE_Orange_Book_Design_Standards.pdf - p.260; C2-1.2.1 Design Flow Rates). Design firm capacity ≥ PHDF, accounting for equipment wear [1] (DOE_Orange_Book_Design_Standards.pdf - p.260; C2-1.2.2 System Hydraulics).

4. **Select Number of Pumps**: Select pumps to achieve PHDF with largest pump out of service (redundancy for EPA Class 1 reliability) [1] (DOE_Orange_Book_Design_Standards.pdf - p.260; C2-1.2.3 Number of Pumps; [3] DOE_Orange_Book_Design_Standards.pdf - p.262; C2-1.2.3 Number of Pumps). For constant speed, base on capacity for minimum scour velocity in force main [3] (DOE_Orange_Book_Design_Standards.pdf - p.262; C2-1.2.3 Number of Pumps). Pumps must handle ≥3-inch solids; suction/discharge ≥4 inches [3] (DOE_Orange_Book_Design_Standards.pdf - p.262; C2-1.2.4 Pump Selection). Evaluate full hydraulic range (new/old pipes, min/max flows, head loss coefficients conservatively) to avoid operation outside manufacturer range [1] (DOE_Orange_Book_Design_Standards.pdf - p.260; C2-1.2.2 System Hydraulics; [2] DOE_Orange_Book_Design_Standards.pdf - p.262; C2-1.2.2 System Hydraulics).

5. **Size Pumps and Force Main**: Select pump capacity (Q) based on PHDF/firm units; balance force main diameter for solids deposition prevention and surge minimization (see C2-3 for sizing) [1] (DOE_Orange_Book_Design_Standards.pdf - p.260; C2-1.2.2 System Hydraulics). Perform cost-benefit analysis; ensure optimum head, velocity, and pump selection [1] (DOE_Orange_Book_Design_Standards.pdf - p.260; C2-1.2.2 System Hydraulics). Provide run timers for flow measurement [3] (DOE_Orange_Book_Design_Standards.pdf - p.266; C2-1.4 Flow Measurement).

6. **Design Wetwell Volume and Geometry**: Calculate minimum volume (V) between pump on/off levels for constant speed pumps: V = (t × Q)/4, where t = minimum time between starts (to prevent excessive cycling), Q = pump capacity (gpm) [2] (DOE_Orange_Book_Design_Standards.pdf - p.262; C2-1.2.5 Wetwells). Correlate volume/pump number to avoid short cycles; provide adequate depth for controls and minimize solids deposition [1] (DOE_Orange_Book_Design_Standards.pdf - p.260; C2-1.2.3 Number of Pumps; C2-1.2.5 Wetwells). Use trench/hopper style (side slopes ≥45°, prefer 60°); reduce free fall, prerotation, vortices; include solids flush provisions (e.g., recycle pipe, periodic drawdown) [2] (DOE_Orange_Book_Design_Standards.pdf - p.262; C2-1.2.5 Wetwells). For raw sewage pre-grit removal, prevent grit settling in discharge lines [3] (DOE_Orange_Book_Design_Standards.pdf - p.266; C2-1.3 Grit, Grease, and Clogging Protection).

7. **Incorporate Reliability and O&M**: Ensure redundancy (pumps, power, controls) for Class 1 reliability; include emergency power (portable/permanent generator sized for peak load) and alarms/telemetry [3] (DOE_Orange_Book_Design_Standards.pdf - p.266; C2-1.8 Reliability; C2-1.8.2 Equipment Redundancy; C2-1.8.3 Emergency Power). Develop O&M manual with pump curves, system curves, startup/shutdown, maintenance, and contingency plans [3] (DOE_Orange_Book_Design_Standards.pdf - p.266; C2-1.7 Operations and Maintenance). Submit as part of engineering report/general sewer plan sequence [9] (DOE_Orange_Book_Design_Standards.pdf - p.33; G1-2.3 General Sewer Plans; Figure G1-1).
