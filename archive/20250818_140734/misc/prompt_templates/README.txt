Folder: prompt_templates

Naming conventions:
- Use a numeric prefix to order templates: 01_, 02_, 03_
- Use kebab-case for names and keep them short and descriptive
- Extension: .md for all templates

Examples:
- 01_engineering-judgement-validation.md
- 02_comparative-analysis.md
- 03_outline-generator.md

How to use with the wizard:
- Place your templates in this folder
- The wizard can list and let you choose a template by number
- The selected template content will be loaded as the query text

Notes:
- Keep template instructions concise and structured
- Prefer explicit Output Format and Constraints sections
