# Wastewater Facility Plan Review Criteria

## Focus Areas
- Compliance with WAC 173-240-060 requirements
- Adequacy of flow and loading projections
- Completeness of alternatives analysis
- Clarity of cost estimates and funding strategy
- Appropriateness of selected treatment technology
- Compliance with water quality standards
- Adequacy of operation and maintenance planning

## Required Sections
- Executive Summary
- Project Background and Description
- Planning Considerations
- Flow and Loading Analysis
- Existing Facility Evaluation
- Future Conditions and Facility Needs
- Alternatives Analysis
- Recommended Alternative
- Implementation Plan
- Financial Analysis
- Operation and Maintenance Requirements
- Environmental Review

## Required Topics
- Service area description and population projections
- Regulatory requirements and permits
- Infiltration and inflow analysis
- Wastewater characterization
- Receiving water quality standards
- Treatment process design criteria
- Hydraulic profile
- Solids handling and disposal
- Cost estimates (capital and O&M)
- Implementation schedule
- Funding strategy
- Environmental impacts

## Citation Requirements
- All regulatory requirements must cite specific WAC/RCW sections
- Population projections must cite official sources (OFM, Census)
- Design criteria must reference accepted standards (Ten States, WEF, etc.)
- Water quality data must cite monitoring reports or studies
- Cost estimates must reference source of unit costs

## Technical Requirements
- Flow projections must include average, maximum month, and peak
- Loading projections must include BOD, TSS, and nutrients if applicable
- Alternatives analysis must include lifecycle cost analysis
- Present worth calculations must use current discount rates
- Design criteria must meet reliability standards
- Treatment process must achieve required effluent quality

## Style Requirements
- Define all acronyms on first use
- Use consistent units throughout (MGD for flow, mg/L for concentrations)
- Include tables for numerical data
- Include figures for process flow diagrams
- Use active voice for recommendations
- Clearly distinguish between existing and proposed facilities

## Custom Patterns
- design_period: design period|planning horizon|20[- ]year
- flow_projections: average daily flow|maximum month|peak hour|peaking factor
- loading_projections: BOD|TSS|nitrogen|phosphorus|loading
- alternatives_analysis: alternatives|options|evaluation criteria
- lifecycle_cost: present worth|lifecycle cost|annual cost
- regulatory_compliance: effluent limits|water quality standards|NPDES
- implementation_schedule: schedule|timeline|phasing
- funding_strategy: funding|financing|rates|fees
- environmental_review: SEPA|environmental assessment|impacts
